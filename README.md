# zeal-solitaire

zeal-solitaire is a solitaire game for the Zeal 8-bit Computer.

It runs at **640×480 @8bpp**

## Rules



## Controls

- Splash: `ENTER` starts the game
- `UP` `DOWN` `LEFT` `RIGHT` (or SNES D-pad): move hand selector around
- `SPACE` (or SNES `B`): reveal selected card
- `X` (or SNES `A`): undo previous card selection
- `P`: pause/resume gameplay music track
- `RIGHT_SHIFT` (or SNES `SELECT`): quit
